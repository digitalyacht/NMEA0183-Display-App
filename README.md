# NMEA0183 Display App

![Screen Shot with iKreate USB Interface.](https://github.com/digitalyacht/NMEA0183-Display-App/blob/main/Screen%20Shots/USB-C+Display%20App.jpg?raw=true)

Welcome to our NMEA0183 Display App repository. This simple but powerful web app, in conjunction with our USB to NMEA Adaptor cables, can display all received Sentences from your NMEA 0183 equipment, send serial commands and store the received data to a log file. 

Written in HTML5, the app uses Web Serial APIs, to access the USB Serial port on your computer at 4800, 38400 115200 or 230400 baud. Simply select the correct COM port and then the web app will Auto-Detect the data baud rate and start receiving data.

This is the world's first multi-platform NMEA 0183 display program and whether you use Windows, LINUX, Mac OSX or Raspbian this simple app (single HTML file) which requires no installation, will allow you to see exactly what NMEA 0183 data is being received.  

If you like this display app, then please check out our [NMEA2000 Display App](https://github.com/digitalyacht/NMEA2000-Display-App) which provides all of this functionality and more for NMEA2000 networks.
